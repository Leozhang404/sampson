# A Statistical Distribution Library

## Currently has the following:

### Distributions

- Bernoulli
- Binomial
- Cauchy
- Gamma
- Normal

### Statistics

- Sample
- T-Tests

### General Functions

- n Choose k
- Factorial
- Gamma
- lnGamma
- Mean
- Standard Deviation
- Stirling Approximation
- Summation
- Nth Power Deviation Summation
- Variance
- Z-Score
